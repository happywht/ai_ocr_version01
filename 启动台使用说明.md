# 专业启动台使用说明

## 🎯 概述

专业启动台是为发票OCR识别工具设计的现代化图形化启动管理界面，提供直观的服务监控和模块管理功能。

## 🚀 快速启动

### 方法一：一键启动（推荐）
双击 `启动专业启动台.bat` 文件，即可直接启动专业启动台。

### 方法二：Python脚本启动
```bash
python 快速启动.py
```

### 方法三：智能选择启动
```bash
python 启动器.py
```
选择"1. 专业启动台UI"选项。

## 🎛️ 功能特性

### 1. 实时服务状态监控
- **OCR服务状态**：实时显示OCR识别服务的运行状态
- **自动检测**：每5秒自动检查服务连接状态
- **快速启动**：一键启动/停止OCR服务

### 2. 独立模块管理
- **发票OCR识别界面**：独立启动和管理发票识别功能
- **字段配置管理器**：独立启动和管理字段配置功能
- **状态追踪**：实时显示各模块的运行状态
- **优雅停止**：支持安全停止各功能模块

### 3. 现代化界面设计
- **专业外观**：采用现代化设计语言
- **响应式布局**：支持窗口大小调整
- **状态指示**：直观的运行状态显示
- **实时时钟**：显示当前系统时间

## 🔧 使用步骤

### 启动OCR服务
1. 打开专业启动台
2. 在"服务状态监控"区域查看OCR服务状态
3. 如果显示"未运行"，点击"启动OCR服务"按钮
4. 等待服务启动完成，状态变为"运行中"

### 启动发票OCR识别功能
1. 确保OCR服务已启动
2. 在"功能模块控制"区域找到"发票OCR识别"
3. 点击"启动"按钮
4. 系统将在新窗口中打开发票识别界面
5. 启动台会显示"运行中"状态

### 启动字段配置管理功能
1. 在"功能模块控制"区域找到"字段配置管理器"
2. 点击"启动"按钮
3. 系统将在新窗口中打开字段配置界面
4. 启动台会显示"运行中"状态

### 停止功能模块
1. 找到需要停止的模块
2. 点击"停止"按钮
3. 模块状态将更新为"未运行"

## 🚨 注意事项

### 1. 依赖要求
- Python 3.7+
- tkinter库（GUI支持）
- umi-OCR服务

### 2. 端口配置
- OCR服务默认端口：1224
- 确保端口未被其他程序占用

### 3. 文件完整性
确保以下文件存在：
- `src/launcher_gui.py` - 启动台主程序
- `src/invoice_gui.py` - 发票识别界面
- `src/field_config_gui.py` - 字段配置界面
- `启动专业启动台.bat` - 一键启动脚本

## 🔍 故障排除

### 问题1：启动台无法启动
**解决方案：**
1. 检查Python是否正确安装
2. 确认tkinter库可用
3. 检查src目录完整性

### 问题2：OCR服务启动失败
**解决方案：**
1. 检查umi-OCR是否已安装
2. 确认端口1224未被占用
3. 手动启动OCR服务

### 问题3：功能模块启动失败
**解决方案：**
1. 确认对应的GUI文件存在
2. 检查依赖库是否完整
3. 查看详细错误信息
